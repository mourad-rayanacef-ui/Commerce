# Sales & Inventory Analytics Platform

A full-stack analytics platform for sales forecasting and inventory management.

## Features
- 📊 Interactive sales dashboard with KPIs
- 📈 Demand forecasting using moving averages
- 📦 Real-time inventory tracking
- ⚠️ Automated low stock alerts
- 🔮 Reorder suggestions based on sales patterns

## Tech Stack
- **Frontend**: React, React Router, CSS
- **Backend**: FastAPI (Python)
- **Database**: PostgreSQL with SQLAlchemy
- **Data Processing**: Pandas, NumPy
- **Deployment**: Docker, (Vercel/Railway optional)
